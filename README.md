## 简介    
1. 首先，安装Docker和Docker Compose。  
2. 最后，一行命令，从空白服务器搭建可直接公开访问的网站：`docker compose up --build -d`，访问`ip:80`即可

>[!tip] 
>linux使用`hostname -I`查看ip, win使用`ipconfig`查看ip; 本地直接输入`localhost`即可访问  
>没有云服务器没关系，可以在本机或虚拟机或 WSL 里面尝试。
